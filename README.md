CAMPUS CONNECT SYSTEM
Campus Connect System is a Python-based mini project developed for MCA students. The system helps manage student placement information      and eligibility details using SQLite database. It includes an admin login module, student registration, and placement eligibility           verification.

Features
*Admin Login
* Student Registration
* Placement Eligibility Check
* SQLite Database Integration

Technologies Used

* Python
* SQLite
* Tkinter
